### ToDo API 

Esse projeto esta em fase de desenvolvimento. 
Segue o padrão MVC e possui algumas mudanças.
---
O projeto esta com o swagger para identificar e testar algumas rotas.
Para rodar localmente: 
- Verifique a cadeia de conexão com o banco de dados SQL Server em appsettings.json
  ```markdown
      "ConnectionStrings": {
        "DefaultConnection": "{Cadeia_Conexao}"
      }
  ```
